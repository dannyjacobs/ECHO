    About

    The ECHO project is an open source hardware/software project. Join us!

    • We have an active “Slack Connect” channel, accessible to anyone with a Slack account. To be invited, email Danny
    • Ask questions on Github Discussions
    • Analysis code on Github
    • Code documentation on ReadTheDocs
    • Hardware also on Github
    • Build logs, flight reports, and other topics are covered in the Memo series
    • ECHO is organized by Professor Daniel Jacobs in the Low frequency Cosmology Lab at Arizona State University.
